== Description ==

Bytes Route is a Digital Adoption Platform that assists businesses in increasing user engagement, encouraging product adoption, and lowering churn rates. By using Bytes Route, creators can design interactive onboarding experiences for their users, ensuring that they grasp the true worth of their digital offerings.

This plugin allows to integrate the Bytes Route onboarding content into WordPress sites.

[Demo video](https://www.youtube.com/watch?v=nb3s8tA8wBA)


== Installation ==

This section describes how to install the plugin and get it working.

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type Bytes Route in the search box and click on search button.
3. Find Bytes Route plugin.
4. Then click on Install Now after that activate the plugin.
5. Open or register your Bytes Route account on https://app.bytesroute.com/login
6. Copy your Bytes Route code snippet
7. Return to the Bytes Route plugin in WordPress and submit the Bytes Route code snippet

For more information, refer to article: [Getting started with Bytes Route](https://www.bytesroute.com/docs/getting-started/getting-started-with-bytes-route.html)

== Frequently Asked Questions ==

= Does Bytes Route require any coding skills to create a web product tour?  =

No, Bytes Route is code-free, which means you do not need to have any coding skills to create a web product tour.

= Is user data stored by Bytes Route during the onboarding process? =

No, Bytes Route does not store any user data. It is cookie-free, so users can forget about cookie policy updates, GDPR, CCPA, and other privacy data regulations. Only usage data is stored.

= What type of businesses can benefit from using Bytes Route for their web product onboarding?  =

Bytes Route can be ideal for SaaS companies, early start-ups focused on fast growth, e-commerce websites, marketers with low access to technical resources, product managers, sales reps, and onboarding managers looking to add an interactive dimension to their traditional onboarding process.

= What can you do with Bytes Route? =

With Bytes Route, you can create various types of digital onboarding tours, including general in-app announcements and step-by-step onboarding processes. It offers multilanguage support and customizable themes to personalize the user experience. Additionally, Bytes Route provides analytics about the performance of web tours, helping you optimize the onboarding experience.

= What does Bytes Route offer to its users? =

Bytes Route is a comprehensive digital onboarding tool with features such as unlimited web domains, web tours, steps per web tours, assists, and active users and also a FREE forever version that allows three active product tours to run concurrently. Users can manage web tours directly from the Chrome extension or the Bytes Route dashboard, and end-users are served all the tours without limitations. The platform also supports custom themes, the option to remove the Bytes Route watermark, and analytics about the performance of the web tours.
